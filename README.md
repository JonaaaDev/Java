# Java Calculator

Application developed in Java Swing using Maven.

## Requirements

*   Java JDK 17+
*   Maven (Optional, scripts include fallback)

## How to Run

### Windows
Double-click `run_windows.bat`

### Linux / macOS
Open terminal and run:
```bash
chmod +x run_unix.sh
./run_unix.sh
```

## Structure
*   `src/main/java`: Source code
*   `pom.xml`: Maven configuration
